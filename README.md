  ## Installation

  To run this application, please do the following installation:
  
  `
  mysql -u root -p
  `

  `
  source db/schema.sql
  `

  `
  npm run seed 
  `

  ## Usage

  `
  use npm start
  `

  Use Insomnia Core to interact with the data
